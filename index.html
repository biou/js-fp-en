Définition

Une méthode de programmation qui préfère l'utilisation de fonctions aux changements d'état.
1e comparaison concrète for -> foreach 

var notes = [ 'do', 'ré', 'mi', 'fa' ];
for (var i = 0; i < 10; i++) {
  console.log(notes[i]);
}
vs 

notes.forEach(function(fruit){
  console.log(fruit);
})

Functions as first class citizen

Fonction d'ordre supérieur

Dans un langage traitant les fonctions comme des "citoyens de 1ère classe", les fonctions peuvent être passées en paramètre (et en retour) d'autres fonctions. 

function octave () {
  return function (note) {
    return note + 8;
  }
}
octave(1) //  9
octave(2) // 10
