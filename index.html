Definition

A programming method that prefers using functions to state changes.
lots of other definitions, we tried to keep it simple here
base concept: function
we will do everything with functions, compose them, etc.
but at the same time trying to avoid state changes
that the main difference with other methods like imperative or OOP

Pure functions

A pure function is a determinist function
without any state change.

It uses only its parameters and local variables.

What for?

By keeping functions pure,
we avoid mutations and thus side effects.
