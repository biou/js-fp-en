MIDI?

  • File format / protocol for music
  • Sequence of messages
  • Minimalist music instrument:
  • Message structure:
    channelsubtypedeltaTimenoteNumber
created in the eighties
func. prog. is good at managing data streams. Midi = streams or files. A file is simply a recording of a stream
main attributes of a message:
channel: 16 midi channels, can play music simultaneously, possible to assign a different music instrument to each channel
subtype: message type, mainly noteOn / noteOff to start and finish playing a note
deltaTime: time between this message and the previous one
noteNumber: number of a note in semitones
