Getting started

Télécharger l'app pour votre plateforme ici:
http://www.sous-anneau.org/pw2016/
ou 

$> git clone git@github.com:vincent/js-fp-tool.git
$> npm i && npm start
Donc vous devriez avoir devant vous quelque chose qui ressemble à ça.
Grosso modo vous avez un petit environement Web avec
- sur la partie gauche un petit éditeur de texte pour écrire du Javascript
- sur la partie droite un debugger, c'est le debugger de Chrome
- et un bouton exécuter qui ... exécute le code
Si vous cliquez dessus sans avoir rien modifié, vous devriez voir le debugger lancé mais arrêté,
car nous avons mis dans le code le mot-clé "debugger", qui permet d'arrêter temporairement l'exécution,
vous pouvez la reprendre en cliquant sur la flèche qui ressemble à "Pause"
