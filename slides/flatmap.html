flatMap()

Applies a function to each element, and
returns a "flattened" list. 


// definition of flatMap for the Array class
// https://gist.github.com/samgiles/762ee337dff48623e729
Array.prototype.flatMap = function (lambda) { 
  return Array.prototype.concat.apply([], this.map(lambda)) 
};
same as map followed by flatten.
Does not exist in JS but very useful. Lots of libraries like lodash have it.
Is proposed to be in the standard for ES7
