map()

Applique une fonction à chaque élément, et
retourne les résultats. 


var partition = [ 42, 24, 91, 14, 27, 34 ];

partition.map(function (note) {
  return octave(note)
})

// [ 50, 32, 99, 22, 35, 42 ]
Utile pour transformer une liste
Ou appliquer un même process avec plusieurs valeurs de départ
