Monade: Either

Soit une valeur, soit une erreur. 

// Création d'un Either
var a = Either.Right(value)
var b = Either.Left(error)

// chaîner
var c = b.flatMap(task)
// task ne sera appliquée que si pas d'erreur

// accéder à la valeur, gérer les erreurs
b.getOrElse(logFailure)
même principe que maybe, éviter de gérer des exceptions ou autres dans une chaîne.
