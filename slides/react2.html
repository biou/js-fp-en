React

one component = one function! 

function HelloWorld (name) {
  function sayHi (event) {
    alert("Hi " + name);
  };
  return {
    <div>
      <a href="#" onClick="{sayHi}">Say Hi</a>
    </div>
  };
};
- component = function, and then composition of function to build the UI (declarative syntax JSX)
- function takes as parameter the state and generates the dom tree
