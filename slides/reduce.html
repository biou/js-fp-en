reduce()

Réduit une liste, en appliquant une fonction à chaque
élément, et retourne le résultat final. 


var liste = [ 1, 2, 3 ];

function somme(acc, e) {
  return acc + e;
}

liste.reduce(somme, 0) // 6
Utile pour transformer une liste en une valeur finale: somme, moyenne, etc.
Ou compter les éléments qui vérifient une condition
