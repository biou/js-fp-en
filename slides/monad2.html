Monads

2 base methods 

// - a constructor, to encapsulate an object in a monad
var a = new Monad("Beethoven")

// - a chain method, to apply a function to the contained objects
// and returns a monad (= flatMap)

a.chain(function(s) {return s.replace("Beethoven", "Mozart")})
// -> Monad("Mozart")
chain = flatMap to avoid encapsulating monads into monads.
If we add flatMap to the Array class, this is then a monad
