Monades

2 méthodes de base 

// - un constructeur, encapsule un objet dans une monade
var a = new Monad("plop");

// - une méthode chain, applique une fonction aux objets contenus 
// et renvoie une monade (= flatMap)

a.flatMap(function(s) {return s.replace("plop", "coin")});
// -> Monad("coin")
méthode chain ou Flatmap
flatMap pour éviter d'encapsuler des monades dans des monades.
Si on rajoute flatMap à la classe Array, c'est alors une monade
