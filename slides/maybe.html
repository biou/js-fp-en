Monade: Maybe

Soit un objet, soit null 

// création d'un Maybe
var a = Maybe.some(object)
var b = Maybe.none()

// chaîner
var c = a.chain(task); 
// task n'a pas besoin de gérer le cas null / undefined

// récupérer la valeur (si nécessaire)
var d = c.orElse(default)
var e = c.isNone()
utile pour éviter de dupliquer des tests de valeurs null ou undefined dans chaque fonction d'une chaîne
la méthode flatMap est aussi appelée chain
