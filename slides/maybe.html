Monad: Maybe

An object or null 

// creation of a Maybe
var a = Maybe.some(object)
var b = Maybe.none()

// chaining
var c = a.chain(task).chain(task2); 
// task and task2 do not need to manage the case null / undefined

// get the value (if needed)
var d = c.getOrElse(default)
var e = c.isNone()
useful to avoid to duplicate testing for null or undefined values in each function of the chain
another name of chain is flatMap
