find()

Applique une fonction à chaque élément, et
retourne le premier pour lequel cette fonction renvoie True. 


var partition = [ 42, 24, 91, 14, 27, 34 ]

partition.find(function (note) {
  return note < 30
})

// 24
Utile pour chercher un élément particulier
Ou savoir si "au moins 1 élément vérifie une condition"
