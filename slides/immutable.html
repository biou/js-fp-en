Immutable.js

  • Une structure de données immuable ne peut pas être modifiée une fois créée.
  • Intérêt: éviter les couplages accidentels, accès à des techniques de mémoisation, égalité
  • JavaScript ne fournit pas d'objet immuable
  • simplifie le test d'égalité de valeur entre 2 objets
  • Immutable.js fournit des structures de données immuables: List, Stack, Map, OrderedMap, Set, OrderedSet et Record.
  • Alternative: mori
Immutable.js développé par Facebook
On peut utiliser Object.freeze en JavaScript, mais l'immuabilité ne fonctionnera qu'au premier niveau.
Mémoisation: cache de résultats de calculs
test d'égalité de valeur, en JS l'égalité entre objets se base sur la référence. avec l'immutabilité, on peut savoir si un objet a été modifié ou non (et par exemple pour react, savoir si on doit mettre à jour la vue ou pas)
