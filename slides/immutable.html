Immutable.js

An immutable data structure
cannot be changed once created.
  • Benefits: prevent accidental coupling, cache, equality
  • Immutable structures in Immutable.js: List, Stack, Map, OrderedMap, Set, OrderedSet and Record.
  • Alternative: mori
Immutable.js developped by facebook
JavaScript does not provide immutable objects
possible to use Object.freeze in JavaScript, but immutability will only work at the first level
Mémoisation: cache of computation results
equality tests of values. In JS, equality between objects is based on the reference. With Immutability, it is possible to know if an object has been modified or not(and for example for react to know if we should update the view or not)
